The display Property

display: inline

The element generates one or more inline element boxes that do not generate line breaks before or after themselves. In normal flow, the next element will be on the same line if there is space.

This is an example of using inline elements for displaying images or icons next to text.

display: inline-block

Inline-block elements are similar to inline elements, but they can have a width and height, as well as padding and margin properties. This makes them useful for displaying elements side by side, without disrupting the flow of text.

This is an example of using inline-block elements for displaying images or icons next to text.

display: block

Block elements take up the full width of their container and start on a new line. They are useful for creating distinct sections of content on a page, such as headings, paragraphs, or lists.

This is an example of using block elements for creating sections of content with a consistent visual style.
